Откуда заполняются поля мерчанта
------------

- Name - заполняется в MerhcantMatcher, на основе очищенного (MerchantNameNormalizer) Description транзакции, при первой встрече такого мерчанта
- DisplayName
    - заполняется в MerchantDisplayNameCommand. Не зашедулена. Последний запуск в октябре 2020.        Логика внутри команды: 
      - найти первые 5000 самых популярных мерчантов без DisplayName
      - найти транзакции для этих мерчантов
      - выставить DisplayName как titleCase от самого часто встречающегося Description среди транзакций этого мерчанта
    - вручную, в схеме мерчантов
- Transactions 
    - заполняется раз в день, в ReportBuilderCommand
    - Увеличивается / уменьшается при Match Transactions / Edit из админки мерчантов.
    
Как перенести транзакции на другого мерчанта
---------------
Выбрать галочкой в админке мерчанта откуда вы хотите перенести.
Нажать "Match Transactions". Дождаться сообщения в чат об 
окончании переноса.
