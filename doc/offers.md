# Офферы в новом дизайне

## Переделка старых офферов

За показ оффера в новом дизайне отвечает переменная "popup". Если она есть:

 - Оффер необходимо рендерить с использованием шаблона "@AwardWalletMain/Offer/_offerPopup.html.twig".
 - Отключить загрузку старых стилей {{ assets.assets_form('css') }}
 - Кнопки выводить с помощью макроса {{ offerMacros.offerButtons(_context) }}
